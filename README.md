# 📚 VoxLibris AI

An AI-powered interactive book library where users upload PDFs and have real-time voice conversations with their books.

## ✨ Overview

VoxLibris AI transforms static PDFs into interactive, voice-driven knowledge experiences.  
Users can upload books, ask questions, request summaries, and have AI respond with natural speech.

This project combines:

- 📖 PDF processing
- 🧠 Large Language Models
- 🔍 Semantic search
- 🎙️ Text-to-Speech (AI voice)
- 🌐 Full-stack web application

---

## 🧠 Features

- 📤 Upload PDF books
- 🔎 Ask questions about book content
- 📝 AI-generated summaries
- 🎧 Real-time AI voice responses
- 💬 Conversational memory
- ⚡ Fast semantic retrieval
- 🔐 Secure user authentication (if applicable)
